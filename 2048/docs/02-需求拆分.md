### M2 核心玩法（游戏层）✅

#### 任务 2.1：Board 类核心算法 ✅

| 字段 | 内容 |
| --- | --- |
| 任务 ID | T-2.1 |
| 优先级 | P0 |
| 实际工时 | 1.5h |

**任务描述**：
- 创建 `src/engine/Board.ts`
- 实现网格状态管理（4×4 CellValue[][]）
- 实现移动/合并算法（旋转到 left 统一处理）
- 实现撤销栈（深拷贝快照，深度 ≤ 10）
- 实现胜负判定

**关键代码结构**：
```ts
export class Board {
  private grid: Grid;
  private history: GridSnapshot[] = [];
  public move(direction: Direction): MoveResult;
  public undo(): boolean;
  public reset(): void;
  private canMoveAnyDirection(): boolean;
}
```

**验收标准**：
- [x] 16 个单元测试全绿
- [x] 移动/合并算法方向正确
- [x] 撤销栈最多 10 步
- [x] 2048 自动判定胜利
- [x] 网格满且无可合并时判定失败
