## 一、项目背景

### 1.1 立项动机
2048 是 Gabriele Cirulli 在 2014 年发布的现象级休闲游戏，规则简洁却深度十足。选择它作为本项目库（Web_Game_01）的第二款产品，原因有三：

1. **技术代表性**：2048 涉及二维数组操作、CSS Grid 布局、撤销栈、动画过渡，是中级前端项目的最佳练手。
2. **复用性强**：项目骨架（game/state/UI 三层架构 + Zustand + shadcn）继承自 snake，可作为后续俄罗斯方块、推箱子等网格类项目的参考模板。
3. **可扩展空间**：经典玩法之上可叠加撤销、步数统计、主题切换、动画、音效等功能，自然适合做"质量优先栈"。

### 1.2 项目定位
- **类型**：2D 网页休闲益智游戏
- **风格**：经典 2048 暖色方块 + 暗黑紫外框（项目设计语言）
- **目标平台**：桌面浏览器（Chrome / Edge / Firefox / Safari）+ 移动端浏览器

## 二、项目目标

### 2.1 学习目标（核心）

| 序号 | 能力 | 衡量标准 |
| --- | --- | --- |
| 1 | 二维数组操作 | 能独立实现网格的移动 / 合并 / 生成 |
| 2 | CSS Grid 布局 | 理解 `grid-template` 与 absolute 浮层结合 |
| 3 | 撤销机制 | 深拷贝快照栈，限制最大深度 |
| 4 | Framer Motion 动画 | 移动 / 合并 / 生成三态过渡 |
| 5 | 状态机 | 5 个游戏阶段（menu/playing/paused/won/over） |
| 6 | 三层架构 | 严格分离 engine（TS 类）/ state（Zustand）/ UI（React） |
| 7 | 触屏输入 | 滑动方向识别 + 30px 阈值 |

### 2.2 产品目标
- ✅ 交付一款可在浏览器中流畅运行的 2048 游戏
- ✅ 实现 PRD-02 中全部 MVP 功能（F-01 至 F-10）
- ✅ 完成至少 4 项 V2 扩展（继续挑战 / 触屏 / 动画 / 音效 / 步数）
- ✅ 可通过 Vercel 一键部署

### 2.3 质量目标

| 指标 | 目标 |
| --- | --- |
| 性能 | 移动响应 ≤ 50ms，动画 60fps 无掉帧 |
| 兼容性 | Chrome 100+ / Edge 100+ / Firefox 100+ / Safari 15+ / 移动端主流浏览器 |
| 代码质量 | TypeScript 严格模式，ESLint 通过，核心算法单测全绿 |
| 体积 | 首屏加载 ≤ 200KB（gzip） |
| 可用性 | 3 秒内可上手，支持键盘 / 触屏 |

## 三、范围定义

### 3.1 ✅ 包含范围

| 模块 | 范围 |
| --- | --- |
| 核心玩法 | 标准 4×4 2048 |
| 控制方式 | 键盘（方向键 / WASD）+ 触屏滑动 |
| 数据存储 | localStorage（最高分 + 音效开关） |
| 撤销 | 最多 10 步回退 |
| UI | 菜单 / 游戏中 / 暂停 / 胜利 / 结束 五个场景 |
| 视觉 | 经典 2048 暖色方块 + 暗黑紫外框 |
| V2 增强 | 动画 / 音效 / 触屏 / 步数统计 / 继续挑战 |

### 3.2 ❌ 不包含范围

| 不包含 | 原因 |
| --- | --- |
| 自定义网格大小（5×5 / 6×6） | PRD 明确排除 |
| 联机对战 | 单机休闲项目 |
| 主题切换（数字 / 字母 / 表情） | 优先级 P2，资源紧张时跳过 |
| 成就系统 | 不在 V1 范围 |
| 网络排行榜 | 无后端依赖 |

## 五、风险评估

| 风险 | 概率 | 影响 | 应对措施 |
| --- | --- | --- | --- |
| 触屏滑动误触 | 中 | 中 | 30px 阈值过滤短距离滑动 |
| Framer Motion layout 性能问题 | 低 | 中 | 仅给 isNew / isMerged 的 tile 加 layout |
| 撤销栈无限增长 | 低 | 低 | 限制最大步数 10 |
| 动画与逻辑不同步 | 中 | 中 | 移动是输入驱动游戏，无需主循环 |
| WinModal 阻断用户继续操作 | 中 | 中 | 用普通 div 半透明遮罩，保留键盘输入 |

## 六、资源投入

### 6.1 人力
- **1 人全栈开发**（前端为主，可独立完成）

### 6.2 时间
- **总工期**：2-3 天（按每天 6-8 小时有效开发计）

### 6.3 技术资源
- Node.js 18+ / npm 9+
- Vite 5.x
- TypeScript 5.x 严格模式
- Zustand 4.x
- Framer Motion 11.x
- shadcn/ui（Radix UI + Tailwind CSS）
- 浏览器开发者工具
- Vercel 账号（免费）

### 6.4 第三方资源
- 图标：Lucide Icons
- 字体：系统默认无衬线字体
- 音效：Web Audio API 实时合成（零素材依赖）
